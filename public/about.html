About Our Game

This game was developed for IN4MATX 285, an Interactive technology Studio for the Master of Human-Computer Interaction and Design program at the University of California, Irvine, taught by Dr. Donald Patterson. This part of the class has the following technical goals:

  1. Be able to rapidly mock up a high-fidelity web based application
  2. Be able to create functionality for the application using client and server side technologies
  3. Be able to debug and identify where failures in the application are occurring so that they can be fixed
